# 标点整理台 · Punctuation Tidy Desk

一个纯前端的标点符号整理工具，帮你在中文和英文文稿之间统一标点风格。

打开 `index.html` 就能用，**无后端、无外部依赖、数据不离开浏览器**。

---

## 功能

- **中英标点互转**：18 种标点双向映射（，。？！：；""''（）【】《》…… 等）
- **空格 / 制表符改逗号**：连续空白归一成一个逗号
- **分号 / 冒号改逗号**：中英文 `; :` 全部归一
- **省略号 / 破折号归一**：`...` → `……`、`--` → `——`（反向同理）
- **合并重复标点**：连续 `，，`、`。。`、`！！`、`？？` 只保留一个
- **清理行首尾空白**：保留换行结构
- **自定义查找替换**：普通文本，不用正则
- **预设切换**：一键"中文文稿" / "英文文稿" 规则组
- **实时统计**：字符数、变更数、行数
- **导出**：
  - 复制到剪贴板
  - 下载为 `.txt`
  - 下载为 `.docx`（Word 可直接打开，纯前端生成，无 Office 依赖）

---

## 使用

1. 双击打开 `index.html`（或拖到浏览器）
2. 左侧"原文"框粘贴或输入文字
3. 右侧"结果"框会实时显示整理后的文字
4. 左侧"规则"区按需调整开关，或点"中文文稿 / 英文文稿"切换预设
5. 点"复制"或"下载 .txt"/"下载 .docx"导出结果

> 想试一下？点左侧的"样例"按钮会自动载入一段混杂标点的样例文本。

---

## 技术

- 纯 HTML + CSS + JavaScript，**单文件 ~28KB**
- 无构建步骤、无 npm、无网络请求
- `.docx` 导出是手写最小 OOXML（CRC32 + ZIP 打包 3 个标准 XML 文件：`[Content_Types].xml` / `_rels/.rels` / `word/document.xml`），详见 `index.html` 内的 `createDocxBlob()`
- 全部数据处理在浏览器本地完成

### 浏览器要求

支持 ES2017+ 的现代浏览器：

- Chrome / Edge 90+
- Firefox 88+
- Safari 14+
