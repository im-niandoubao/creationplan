# 摸鱼 Chrome 扩展 1.1.0

## 安装

1. 打开 Chrome：`chrome://extensions/`
2. 打开右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择这个文件夹：`moyu-extension`

## 使用

点击工具栏里的“摸鱼”，填入工作网址、娱乐网址和静止秒数。

输入内容会自动暂存。复制第一个网址后，即使 popup 因为你切出去复制第二个网址而关闭，重新打开扩展时前一个网址也还在。

推荐流程：

1. 点“设置窗口”：工作页和娱乐小窗会同时正常显示。
2. 手动拖拽娱乐小窗，调整到合适大小和位置。
3. 点“启动”：进入自动模式，小窗会直接保持显示。

自动模式里，启动后小窗会直接显示；任意网页里有鼠标或键盘活动后，娱乐小窗会最小化并静音，然后回到工作页。之后工作页没有鼠标或键盘活动达到设定秒数后，会恢复娱乐小窗并取消静音。运行态小窗里键盘和鼠标都会被捕捉，一碰就收回小窗。

娱乐小窗使用 Chrome 的极简 popup 窗口，没有地址栏和标签栏。手动拖拽娱乐小窗调整位置或大小时，扩展会自动记录；下一次自动恢复时会使用这个位置和尺寸。

“停止”会停止自动模式并关闭娱乐小窗。

## 限制

Chrome 扩展不能保证系统级置顶。这个版本使用极简 Chrome popup 小窗口，所以没有浏览器地址栏，但不会像桌面应用那样永远悬浮在所有软件之上。
